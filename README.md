# Text Generation using GPT-2

## Objective
This project focuses on generating meaningful text using GPT-2 and improving output using a custom dataset.

## Dataset
A small dataset of motivational and self-reflection quotes created manually.

## Approach
- Loaded pre-trained GPT-2 model
- Fine-tuned it using custom dataset
- Compared results before and after training

## Results
The model generated more relevant and meaningful text after fine-tuning.
